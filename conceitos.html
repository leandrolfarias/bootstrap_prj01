Conceitos em accordion

Front-End é a parte visual de um site, ou seja, todos os elementos de interação com o usuário. Quem trabalha com Front End é responsável por desenvolver por meio do código uma interface gráfica e, normalmente, com as tecnologias base da Web: HTML, CSS e JavaScript.

Back-End é a estrutura que liga um banco de dados à interface visual da aplicação ou site. Como o próprio nome sugere, é o que tem "por trás" de uma aplicação. Como exemplo, um perfil de uma rede social, como Instagram, Whatsapp, Facebook e outros, contém as informações dos usuários armazenadas em um banco de dados. Quando fazemos o login nesses sites ou aplicativos, estamos interagindo com a parte visual (Front-End), que se comunicará com a parte estrutural de dados (Back-End). O Back-End trabalha em boa parte dos casos fazendo a ponte entre os dados que vem do navegador rumo ao banco de dados e vice-versa, sempre aplicando as devidas regras de negócio, validações e garantias num ambiente restrito ao usuário final (ou seja, onde ele não consegue acessar ou manipular algo).

Full Stack developer é quem trabalha com Front End e Back End. Em resumo, é essa a definição. Um desenvolver que trabalha tanto na parte visual do site ou aplicação, quanto na parte de estruturação do banco de dados.
